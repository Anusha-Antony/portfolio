Doctor Appointment System

Developed a full-stack web-based Doctor Appointment System designed to simplify and automate the healthcare appointment process. The platform allows patients to register, log in, view doctor availability, and book appointments online in real time. Doctors can manage schedules, approve appointments, and reduce manual workload.

The system integrates SMS notifications using the Twilio API to send appointment confirmations and reminders, helping minimize no-shows. Built using PHP and MySQL with a responsive user interface, the project focuses on usability, data accuracy, and efficient appointment management.

BMI Calculator

Designed and developed a responsive BMI Calculator web application that enables users to calculate their Body Mass Index based on height and weight inputs. The system provides instant feedback along with meaningful health insights and personalized food suggestions to support weight management and healthy lifestyle choices.

Emphasis was placed on a clean user interface, intuitive navigation, and real-time result display. The project demonstrates strong front-end development skills, user-focused design principles, and practical application of health-related calculations.
